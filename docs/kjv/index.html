A King James Version (KJV/kjv) plain text bible which was reformatted from a Gutenberg text bible. Goals of reformatting included 1) easier random access using a file manager/explorer, 2) facilitate machine reading of the scriptures using verse per line format, 3) eventually get it into a format used by the SWORD project.

To minimize accidental errors during the reformatting, some software aids were used to compare the Gutenberg with BibleGateway. The table below divides the results of comparison into a few categories.

Personal use reading of the repo was done initially by downloading/cloning content to the local [mobile] device and then unzipping or extracting the content. Then a file manager/explorer opened text files for the desired chapter. The chapter is selected in the kjv folder by clicking on a book folder and then a chapter file number.

How to download? In the past, a download button was remembered on github. But recently this was no longer observed on a mobile device browser. Thus, another web site was used to download the repo to the local device.
