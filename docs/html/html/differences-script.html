Differences from script

Differences reported from the script (S/0) are summarized here. Reports for each chapter (not this file) puts a hyphen (-) after the verse number if no differences were observed. This file is a summary though, so it skips the hyphens and only reports observed differences. Most of the differences were believed to be due to upper/lower case differences because because the browser changed case using html/js/css after the server sent the text. For example, the word 'Lord' sent by biblegateway might be changed to LORD in the browser by the html/js/css. The shell script only looked at the network text and didn't attempt html/js/css operations. The most common difference of (Lord - LORD) was omitted because it was frequent. Some of the differences below also may just be similar case differences.
